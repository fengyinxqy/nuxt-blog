# My Blog

简约大气的个人博客，使用 Nuxt 4 + Nuxt Content v3 构建。

## 功能特性

- 📝 Markdown 内容管理
- 🎨 清爽白色 + 蓝色系主题
- 🔍 全文搜索（Cmd+K）
- 🏷️ 分类/标签系统
- 💬 Giscus 评论系统
- 📱 响应式设计

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 技术栈

- Nuxt 4
- Nuxt Content v3
- Tailwind CSS
- Vue 3 Composition API
- TypeScript

## 部署

支持部署到 Vercel 或 Netlify。
